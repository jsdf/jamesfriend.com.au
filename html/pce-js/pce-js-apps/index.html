PCE Macplus emulator running Mac OS System 7

— PCE.js — a hack by James Friend
PCE.js is a browser port of Hampa Hug's excellent PCE emulator, using Emscripten.

This demo emulates a Mac Plus with a bunch of abandonware applications and games to check out.
