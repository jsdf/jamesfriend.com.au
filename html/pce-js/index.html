PCE.js Mac Plus emulator running Mac OS System 7 — a hack by James Friend

PCE.js emulates classic computers in the browser.

This is a simulation of a Classic Macintosh from 1984, running System 7.0.1 with MacPaint, MacDraw, and Kid Pix. If you want to try out more apps and games see this demo.

It's based on Hampa Hug's excellent PCE emulator, which I ported to the web.

Please be patient as you may see a few seconds of black screen upon loading, but it's worth it.
