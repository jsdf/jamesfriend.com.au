Basilisk II Classic Mac emulator

This is the Basilisk II Classic Mac emulator running Mac OS System 7, with SimCity 2000 and Bungie Software's Marathon. A hack by @ur_friend_james

This emulator should work in Chrome 69 and Chrome 72+ (but not Chrome 70, due to a bug). It should work in Firefox if you enable the javascript.options.shared_memory flag in the about:config page.
NOTE: your browser is missing a required feature for this emulator. There is an alternate version which should work for you here.
